## build

1. install node 

[https://nodejs.org](https://nodejs.org)

2. install go

[https://golang.org/](https://golang.org/)

3. install statik

 statik is a tool to embed static web files into Go binary 

```sh
go get github.com/rakyll/statik
```

4. build

```sh
build
```
